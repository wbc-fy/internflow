# InternFlow

InternFlow is an AI-powered internship application assistant designed to help students analyze job descriptions, match resumes with internship requirements, generate application suggestions, and manage application progress.

## Features

- Analyze internship job descriptions
- Match resume content with job requirements
- Identify matched and missing skills
- Generate resume improvement suggestions
- Generate draft application messages
- Provide a FastAPI backend API
